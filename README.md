# apollo-laravel

### Install PHP dependencies
```
composer install
```
### (Optional - if to truncate the randoms & breakdowns tables)
```
php artisan migrate:fresh
```

### (Install JavaScript dependencies)
```
npm install
```

### Serve API (http://127.0.0.1:8000)
```
php artisan serve
```
### Compile front-end
```
npm run watch
```
